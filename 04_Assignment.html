Q.6 How can the style/class of an element be changed? 

    Changing style
      1. Inline styles: Use the style property to change individual styles.
      - element.style.property = value;
      - Example: document.getElementById("example").style.color = "red";

      2. CSS styles: Use the style.cssText property to change multiple styles at once.
      - element.style.cssText = "property1: value1; property2: value2;";
      - Example: document.getElementById("example").style.cssText = "color: red; font-size: 24px;";

      Changing Classes:
      1. Add a class: Use the classList.add() method.
          - element.classList.add("className");
          - Example: document.getElementById("example").classList.add("example-class");
      2. Remove a class: Use the classList.remove() method.
          - element.classList.remove("className");
          - Example: document.getElementById("example").classList.remove("example-class");
      3. Toggle a class: Use the classList.toggle() method.
          - element.classList.toggle("className");
          - Example: document.getElementById("example").classList.toggle("example-class");
      4. Replace a class: Use the classList.replace() method (not supported in older browsers).
          - element.classList.replace("oldClassName", "newClassName");
          - Example: document.getElementById("example").classList.replace("old-class", "new-class");

      Before changing styles or classes, you need to select the element(s) you want to modify. You can use:

      1. document.getElementById("id") for a single element by ID.
      2. document.getElementsByClassName("className") for multiple elements by class name.
      3. document.querySelectorAll("selector") for multiple elements by CSS selector.

      Remember to replace "example" and "className" with your actual element ID and class names.

Q.7 How to read and write a file using JavaScript? 

    Reading and writing files in JavaScript can be achieved using various methods, depending on the environment and requirements.
    Here are some common approaches:

    Client-side (Browser):
    
    1. FileReader and FileWriter APIs: For reading and writing files locally on the client-side, you can use the FileReader and FileWriter APIs.
     However, these APIs have limited support and security restrictions.
    
    Server-side (Node.js):
    
    1. fs (File System) module: Node.js provides the built-in fs module for reading and writing files. You can use methods like:
        - fs.readFile() and fs.writeFile() for asynchronous operations.
        - fs.readFileSync() and fs.writeFileSync() for synchronous operations.
    2. fs.promises: Node.js 14+ introduces fs.promises, allowing you to use async/await syntax for file operations.
    3. Streams: You can use Node.js streams to read and write files in a more efficient and scalable way.

Q.8 What are all the looping structures in JavaScript? 

        1. for loop: Allows you to run a block of code for a specified number of iterations.

        for (var i = 0; i < 10; i++) {
          console.log(i);
        }
        
        2. while loop: Runs a block of code as long as a specified condition is true.
        
        var i = 0;
        while (i < 10) {
          console.log(i);
          i++;
        }
        
        3. do-while loop: Similar to the while loop, but the code block is executed at least once before the condition is checked.
        
        var i = 0;
        do {
          console.log(i);
          i++;
        } while (i < 10);

        4. forEach loop (used with arrays): Executes a function for each element in an array.
        
        [1, 2, 3, 4, 5].forEach(function(element) {
          console.log(element);
        });
        
        5. for...in loop (used with objects): Iterates over the properties of an object.
        
        var obj = {a: 1, b: 2, c: 3};
        for (var prop in obj) {
          console.log(obj[prop]);
        }
        
        6. for...of loop (used with iterable objects like arrays, strings, etc.): Similar to foreach, but allows for more flexibility.
        
        for (var element of [1, 2, 3, 4, 5]) {
          console.log(element);
        }
        
    
    
Q.9 How can you convert the string of any base to an integer in JavaScript? 

    
      In JavaScript, you can convert a string of any base to an integer using the built-in parseInt() function.
      The parseInt() function takes two arguments:

      1. The string to be converted
      2. The base of the number in the string (optional, defaults to 10)
      
      Here's the syntax:
      
      parseInt(string, base)
      
      For example:
      parseInt('1010', 2) // Output: 10
      
    

  
Q.10 What is the function of the delete operator? 

  
    The delete operator in JavaScript is used to remove a property from an object, or to remove a element from an
    array.
  
    In JavaScript, the delete operator is used to:
      
    1. Remove a property from an object: It deletes a property and its value from an object.
    2. Remove an element from an array: It removes an element from an array, leaving a hole (undefined) in its place.
      
    Syntax:
      
    delete object.property
    delete array[index]
      
    Examples:
      
    1. Delete object property:
      
    const person = { name: 'John', age: 30 };
    delete person.age;
    console.log(person); // Output: { name: 'John' }
      
    1. Delete array element:
      
    const arr = [1, 2, 3, 4, 5];
    delete arr[2];
    console.log(arr); // Output: [1, 2, undefined, 4, 5]

Q.13 How can a page be forced to load another page in JavaScript? 

  
    There are several ways to force a page to load another page in JavaScript:

    1. window.location: Set the window.location property to the URL of the new page.
    
    window.location = '(link unavailable)';
    
    1. window.location.href: Set the window.location.href property to the URL of the new page.
    
    window.location.href = '(link unavailable)';
    
    1. window.location.replace(): Use the window.location.replace() method to replace the current page with a new one.
    
    window.location.replace('(link unavailable)');
    
    1. window.location.assign(): Use the window.location.assign() method to load a new page.
    
    window.location.assign('(link unavailable)');
    
    1. window.open(): Use the window.open() method to open a new page in the current window or a new tab.
    
    window.open('(link unavailable)', '_self'); // Current window
    window.open('(link unavailable)', '_blank'); // New tab

    Also, be aware of the differences between these methods:

    - replace() and assign() will replace the current page's history entry, while href and open() will create a new history entry.
    - _self will load the new page in the current window, while _blank will load it in a new tab.
  


  
Q.14 What are the disadvantages of using innerHTML in JavaScript? 

  
Using innerHTML in JavaScript has several disadvantages:

    1. Security risks: innerHTML can introduce cross-site scripting (XSS) vulnerabilities if user-generated content is not properly sanitized.
    2. Performance issues: Frequent updates to innerHTML can cause the browser to re-render the entire page, leading to performance issues.
    3. Loss of references: When updating innerHTML, all existing references to child elements are lost, which can cause issues with event listeners and other attached data.
    4. Removal of event listeners: innerHTML removes all event listeners attached to child elements, requiring re-attachment after updating.
    5. Inability to append content: innerHTML replaces the entire content, making it difficult to append new content without overwriting existing content.
    6. Browser inconsistencies: innerHTML may behave differently across browsers, leading to compatibility issues.
    7. No support for SVG or XML: innerHTML only works with HTML content, not SVG or XML.
    8. Difficult debugging: Errors in innerHTML content can be challenging to debug due to the lack of clear error messages.
    
    To avoid these disadvantages, consider using alternative methods like:
    
    - createElement and appendChild for adding new elements
    - textContent or innerText for updating text content
    - insertAdjacentHTML for inserting HTML content without replacing existing content
    - Template literals or DOM parsing libraries for safer and more efficient HTML manipulation.
